# vibe-code-projects

A collection of personal projects.

## Projects

- **[speakeasy](./speakeasy)** — fully-local, privacy-first voice dictation for macOS. Press a
  hotkey, speak, and your words are typed into any app. On-device speech-to-text (NVIDIA Parakeet,
  Apple-Silicon accelerated), automatic filler-word cleanup, zero cloud, zero telemetry. A
  privacy-hardened fork of [Handy](https://github.com/cjpais/Handy) (MIT).
